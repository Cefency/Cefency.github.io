安装Package Control

使用快捷键 ctrl + `打开Sublime的控制台 ,或者选择 View > Show Console

安装代码点这里获取

安装MarkdownEditing

使用ctrl + shift + p进入Sublime命令面板，输入package install在列表中选择install package。可能会有一些延时。然后找到MarkdownEditing回车安装。

安装OmniMarkupPreviewer

OmniMarkupPreviewer预览渲染效果。

ctrl + alt + o: 在浏览器中预览
